Guardian Hostel Management System - README

Project Title:
Guardian Hostel Management System

Introduction:
This is a web-based application developed using PHP and MySQL to manage hostel activities like student registration, room allocation, payment, and complaints.

Objective:
- Automate hostel management
- Maintain student records
- Manage room allocation
- Track payments
- Handle complaints

Technologies Used:
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Tools: XAMPP, VS Code

Modules:
1. Student Module
2. Admin Module
3. Room Management
4. Payment Module
5. Complaint Module
6. Login & Security
7. Reports

Workflow:
Student -> Register -> Login -> Request Room -> Admin Approves -> Room Allotment -> Payment -> Complaint

Features:
- User-friendly interface
- Online payment
- Real-time data
- Secure login system

How to Run:
1. Install XAMPP
2. Copy project to htdocs
3. Create database in phpMyAdmin
4. Import SQL file
5. Run on browser: http://localhost/project_name
